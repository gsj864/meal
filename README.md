# 오늘의 급식

학교 급식 식단을 쉽고 빠르게 확인하는 웹앱입니다.  
나이스교육정보개방포털(NEIS) Open API를 사용합니다.

## 기능

### 학교 선택
- **시/도** → **학교급**(초·중·고) → **학교명** 검색 후 선택
- 선택한 학교는 브라우저(localStorage)에 저장되어 다음 접속 시 자동 적용

### 보기 방식
- **일간**: 선택한 날짜 하루의 아침/점심/저녁 식단
- **주간**: 해당 주(월~일) 7일을 가로 한 줄로 표시 (모바일: 가로 스크롤, PC: 7열 그리드)
- **월간**: 해당 월 전체를 1주일씩 가로 행으로 표시 (각 주 7일 가로 배치)

### 날짜 선택
- **날짜 입력(달력)** 으로 원하는 날짜 선택
- 일간/주간/월간에 따라 선택한 날짜가 포함된 하루·주·월이 표시됨

### 알레르기 정보
- 메뉴별 알레르기 유발 식재료 **번호** 표시 (토글로 켜기/끄기)
- 번호 **클릭** 시 상세 팝오버 (번호별 식재료명: 난류, 우유, 대두 등)

### 반응형
- 모바일 우선, PC·큰 화면에서는 넓은 레이아웃 (최대 92vw)
- 일간 보기: PC에서 아침/점심/저녁 3열 배치

## 실행 방법

1. **로컬에서**
   - `index.html`을 브라우저로 열거나
   - CORS 이슈 시 로컬 서버 실행 후 접속:
     ```bash
     npx serve .
     ```
     또는 `python -m http.server 8000` 후 `http://localhost:8000` 접속

2. **배포**
   - GitHub Pages, Netlify 등에 올려 사용 가능
   - 나이스 API가 브라우저 CORS를 허용하지 않을 수 있어, 그 경우 로컬 서버나 백엔드 프록시가 필요할 수 있습니다.

## 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 화면 구조 (헤더, 날짜·보기 탭, 급식 카드, 학교 선택·알레르기 팝오버) |
| `style.css` | 반응형 스타일 (모바일 우선, PC 넓은 레이아웃) |
| `app.js` | NEIS API 연동, 학교 검색/선택, 일·주·월 급식 로드, 알레르기 상세 |

## API 키

`app.js` 상단의 `NEIS_KEY`가 나이스 포털 인증키입니다.  
직접 발급한 키로 교체해 사용하세요.  
[인증키 신청](https://open.neis.go.kr/portal/guide/actKeyPage.do)

## 기술 스택

- **프론트**: HTML, CSS(Flex/Grid), JavaScript
- **API**: NEIS Open API (schoolInfo, mealServiceDietInfo)
- **저장**: localStorage (선택 학교)
